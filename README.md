## 项目部分思路

- 如何设置token

> username + 密码的md5编码

- 如何返回静态资源URL

> 1. 通过`gin`获取`ip`
> 2. 拼接端口和资源路径

- 如何处理关注关系和点赞关系

> 创建一个关系表，记录双方的关系

- 如何避免重复创建关系

> 在关系表中设置`cancel`字段，作为取消标识，避免重复创建

- 如何截取视频封面

> 通过go调用命令行工具`ffmpeg`
